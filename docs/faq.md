# ❓ 자주 묻는 질문 (FAQ)

## 💰 비용이 드나요?

**OpenAI API 비용만 발생합니다.**

- 음성 인식 (Whisper): ~$0.006/분
- TTS: ~$0.015/1000자
- 하루에 10번 정도 사용하면 약 $0.10 (100원 정도)

> 💡 Claude Code는 Claude Pro 구독($20/월)에 포함되어 있어요!

---

## 🛡️ 보안은 괜찮나요?

**네, 안전합니다!**

- 모든 데이터는 **내 맥북에서만** 처리됩니다
- OpenAI에는 음성 데이터만 전송 (노트 내용 X)
- Tailscale VPN으로 안전한 네트워크 연결
- Cloudflare 터널은 일시적 URL (재시작마다 변경)

---

## 📱 아이패드에서도 되나요?

**네! 똑같이 사용 가능해요.**

Safari에서 HTTPS URL 접속 → 홈화면에 추가

---

## 🔒 집 밖에서도 되나요?

**Tailscale 덕분에 어디서든 가능해요!**

1. iPhone에 Tailscale 앱 설치
2. Mac과 같은 계정으로 로그인
3. HTTPS URL로 접속

카페, 회사, 어디서든 내 Obsidian과 대화할 수 있어요.

---

## 🤖 Siri처럼 "Hey Siri"로 시작할 수 있나요?

**iOS 단축어로 비슷하게 만들 수 있어요!**

하지만 웹앱을 열고 녹음 버튼을 누르는 방식이 더 안정적입니다.
(단축어 설정은 고급 사용자용)

---

## 💤 맥북 닫아도 되나요?

**네, 닫아도 됩니다!** 단, 설정이 필요해요:

```bash
# 잠자기 방지 설정
sudo pmset -c sleep 0
sudo pmset -c disksleep 0
```

그리고 **충전기를 연결**하세요. 이렇게 하면:
- ✅ 덮개 닫아도 서버 계속 작동
- ✅ 조용히 책상 한쪽에 두면 OK

> 💡 외부 모니터 연결하면 더 안정적이에요!

---

## 🔄 URL이 자꾸 바뀌어요. 왜죠?

**Cloudflare Quick Tunnel의 특성입니다.**

무료 버전은 터널을 재시작할 때마다 URL이 바뀌어요.

고정 URL을 원하면:
- Cloudflare 계정 생성
- 도메인 연결 (유료)

---

## 📝 어떤 명령을 할 수 있나요?

**거의 모든 걸 말로 할 수 있어요!**

예시:
- "오늘 회의 내용 노트로 정리해줘"
- "지난주에 쓴 일기 찾아줘"
- "할 일 목록에 장보기 추가해줘"
- "내 프로젝트 폴더 구조 알려줘"
- "이 아이디어 Daily Notes에 저장해줘"

Claude가 Obsidian Vault 전체에 접근할 수 있어서, 
파일 생성, 수정, 검색, 요약 모두 가능합니다!
